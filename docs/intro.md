# Physical AI & Humanoid Robotics: From Simulation to Reality

Welcome to the comprehensive textbook for Physical AI & Humanoid Robotics! This 13-week curriculum takes you from fundamental concepts to building autonomous humanoid systems, combining simulation-first learning with real-world applications.

## üìö Course Overview

This textbook is structured into 5 progressive modules that build upon each other:

- **Module 0**: Getting Started (Weeks 1-2) - Course orientation and development environment setup
- **Module 1**: The Robotic Nervous System - ROS 2 (Weeks 3-5) - Master ROS 2 fundamentals for distributed robotic systems
- **Module 2**: The Digital Twin - Gazebo & Unity (Weeks 6-7) - Create physics-accurate simulations for rapid prototyping
- **Module 3**: The AI-Robot Brain - NVIDIA Isaac (Weeks 8-10) - GPU-accelerated perception, navigation, and learning
- **Module 4**: Vision-Language-Action Models (Weeks 11-12) - Natural language control and multimodal interaction
- **Module 5**: Capstone Project - Autonomous Humanoid (Weeks 13-14) - Integrate all modules into a functioning autonomous system

## üõ†Ô∏è Hardware Requirements

This curriculum accommodates three different hardware tiers to suit various budgets and learning objectives:

- **Proxy (Simulation Only)**: 16GB RAM, Quad-core CPU - Sufficient for Modules 0-2
- **Miniature ($1.5k-2k)**: RTX 3060+ GPU, 32GB RAM - Required for Modules 3-4
- **Premium ($90k+)**: Physical AI Edge Kit with Jetson Orin Nano - Required for Module 5

For detailed setup instructions, see our [Hardware Setup Guide](/hardware-guide).

## üìñ Learning Approach

Each chapter follows a consistent structure:
- **Prerequisites**: What you need to know before starting
- **Learning Outcomes**: What you'll be able to do after completing

- **Code Examples**: Downloadable Python/C++ files with documentation
- **Practical Exercises**: Hands-on activities with validation scripts
- **Assessment Checkpoints**: Knowledge checks with immediate feedback
